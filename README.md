# BookSelling Website(책판매 웹사이트-만들기)


## Environment(개발환경)
### front-end(프론트앤드)
- React.js
- html
- postcss
- yarn
### back-end(백앤드)
- localstorage
###API
- Google book api


## Service(제공 기능)
- google book api : search book with query(keyword). 
- 좋아요 기능
- 장바구니기능
- 피드백 부분, 별점기능
- 리뷰 정적 데이터로 관리(구현 제대로 못함)




## UI(최종 완성본 UI)
1. main page(홈페이지)
2. search page(검색 페이지)
3. like page(좋아요 페이지)
4. cart page(장바구니 페이지)
